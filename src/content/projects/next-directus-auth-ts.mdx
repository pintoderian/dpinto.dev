## Introducción a un Proyecto de Dashboard con Next.js y Directus

Debido a que tenía varios proyectos que requerían un panel de administración o dashboard, me vi en la necesidad de crear una plantilla base que utilizara Directus como backend y Next.js como frontend.&#x20;

**¿Por qué elegí Directus?** Porque facilita la creación de tablas relacionales, lo que permite un manejo de datos rápido y eficiente. Además, al ser un CMS headless, Directus proporciona un sistema de autenticación vía API, lo cual es ideal para este tipo de proyectos. Y en cuanto a Next.js, es una elección obvia debido a sus funciones integradas, como los middleware, y las recientes adiciones de server actions, que funcionan de maravilla con el SDK de Directus.

### Páginas Principales

* **Página de Autenticación:** Aquí es donde tus usuarios se registran e inician sesión de manera segura.
* **Página de Dashboard:** Es el corazón del proyecto, donde se muestra toda la información relevante de manera clara y ordenada.

### Características Destacadas

Este proyecto no es solo una simple plantilla; es una base sólida que te ayudará a construir un dashboard web moderno sin complicaciones. Estas son algunas de las cosas que lo hacen genial:

* **Estilo atractivo y fácil de personalizar:** Gracias a Tailwind CSS, puedes cambiar el diseño según las necesidades de tu proyecto sin complicarte.
* **Autenticación segura:** Con NextAuth.js, tus usuarios pueden estar tranquilos de que sus datos están protegidos.
* **Gestión de contenido dinámica:** Directus te permite manejar el contenido de manera flexible, lo que es perfecto para adaptarse a diferentes necesidades.
* **Configuración inicial simplificada:** Todo está listo para que empieces a trabajar, lo que te ahorra tiempo y dolores de cabeza.

### Tecnologías que Utilizamos

Nos apoyamos en algunas de las tecnologías más modernas y útiles para hacer tu vida más fácil:

* **Next.js:** El framework de React ideal para aplicaciones web.
* **Tailwind CSS:** Un framework de diseño CSS que te permite crear estilos bonitos sin esfuerzo.
* **Shadcn:** Componentes visualmente atractivos construidos con Radix UI y Tailwind CSS.
* **next-auth:** Para gestionar la autenticación y autorización de usuarios.
* **next-themes:** Para cambiar de tema de manera sencilla en Next.js.
* **TypeScript:** El lenguaje de programación que te ayuda a escribir código más seguro y eficiente.
* **@directus/sdk:** Cliente para trabajar con la API de Directus.
* **zustand:** Una solución ligera para la gestión de estado en React.
* **zod:** Para la validación de esquemas en TypeScript.
* **Husky:** Para asegurarte de que todo está en orden antes de hacer un commit o push en Git.
* **Prettier:** Para mantener tu código limpio y bien formateado.
* **ESLint:** Para detectar problemas en tu código antes de que se conviertan en errores.

### Requisitos

Antes de comenzar, asegúrate de tener instalados:

* **Node.js:** versión 18.8.0
* **npm:** El gestor de paquetes para JavaScript

Para más detalles sobre cómo usar y personalizar este proyecto, puedes revisar el archivo Readme en el repositorio. Ahí encontrarás instrucciones completas y fáciles de seguir.

Este proyecto está pensado para ser una base flexible y eficiente. ¡Es perfecto para que te concentres en lo realmente importante: crear algo increíble sin preocuparte por la configuración inicial!
