## Codegea: Plataforma de Servicios Digitales

Primero que nada, **Codegea** es mi propia web donde ofrezco una variedad de servicios, incluyendo desarrollo web, software, aplicaciones, migración de datos, web scraping y más. Esta plataforma es un espacio para conectar con clientes y mostrar mi experiencia en diferentes áreas de la tecnología.

## Integración de Publicidad con DoubleClick for Publishers

Trabajé en la integración de publicidad con DoubleClick for Publishers, de manera subcontratada por un proveedor que colaboraba con ADOPS para el sitio web de La Nación de Costa Rica ([https://www.nacion.com/](https://www.nacion.com/)). Este proyecto utilizó tecnologías como REACT y ARC CMS, el mismo sistema de gestión de contenido utilizado por The Washington Post.

## Actualización del Core del Sitio Web de El Universo

También actualicé todo el core del sitio web de El Universo, realizando una transición de REACT 16 a REACT 18 y de NODE 14 a NODE 18. Este proceso implicó actualizar la mayoría de los paquetes npm y los distintos componentes que gestionan la página, asegurando que el sitio estuviera a la vanguardia en términos de tecnología y rendimiento.
