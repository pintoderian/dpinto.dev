## Desarrollo de Soluciones Digitales para [eluniverso.com](https://eluniverso.com)

### Creación de Componentes en React con ARC PUBLISHING

En mi trabajo para eluniverso.com, uno de los proyectos clave fue la creación de componentes en React utilizando ARC PUBLISHING. Esta tecnología permitió una integración fluida y eficiente con el sitio web, mejorando la experiencia del usuario y la gestión del contenido.

### Diseño y Desarrollo de Micrositios

Además, diseñé y desarrollé micrositios utilizando Next.js y Tailwind CSS. Estos micrositios aportaron versatilidad y dinamismo al contenido del sitio, permitiendo una presentación más atractiva y funcional.

### Desarrollo de Liveblog en Tiempo Real

Un logro notable fue el desarrollo de un liveblog en tiempo real con Strapi. Este sistema permitía mostrar noticias con un badge de "live" en Google, actualizando las notas en tiempo real mediante websockets, garantizando una cobertura continua y actualizada.

### Mejora de la Experiencia de Usuario

Para mejorar la experiencia de usuario, adapté componentes utilizando Tailwind CSS. Esto aseguró una interfaz atractiva y funcional, adaptada a las necesidades del usuario final.

### Implementación de Metodología SCRUM

Implementé la metodología SCRUM para gestionar y desplegar tareas de manera eficiente. Esto optimizó el flujo de trabajo y mejoró la coordinación del equipo, asegurando que los proyectos se completaran a tiempo y dentro del presupuesto.

### Configuración y Mantenimiento de Servidores

También configuré y mantuve servidores utilizando Nginx. Además, migré sitios web a Digital Ocean, lo que mejoró significativamente la escalabilidad y eficiencia del sistema.

### Integración de Publicidad con DoubleClick for Publishers

En cuanto a la publicidad, integré DoubleClick for Publishers (DFP) con Piano y mantuve los módulos publicitarios. Esto aseguró una monetización efectiva del contenido, maximizando los ingresos publicitarios del sitio.

### Desarrollo de APIs y Gestión de Contenido

Desarrollé APIs utilizando tecnologías como Express, TypeScript, Jest, Husky y Prettier, garantizando un código limpio y bien estructurado. Además, implementé el sistema de gestión de contenido sin cabeza (Headless CMS) Directus, optimizando los procesos empresariales de la compañía y permitiendo una gestión de contenido más flexible y eficiente.
