### Desarrollo de Soluciones Digitales para eluniverso.com

En mi trabajo para eluniverso.com, desarrollé una serie de soluciones digitales innovadoras utilizando tecnologías modernas. Uno de los proyectos clave fue la creación de componentes en React utilizando ARC PUBLISHING, lo cual permitió una integración fluida y eficiente con el sitio web.

Además, diseñé y desarrollé micrositios utilizando Next.js y Tailwind CSS, aportando a la versatilidad y dinamismo del contenido del sitio.&#x20;

Un logro notable fue el desarrollo de un liveblog en tiempo real con Strapi. Este sistema permitía mostrar noticias con un badge de "live" en Google, actualizando las notas en tiempo real mediante websockets, lo que garantizaba una cobertura continua y actualizada.

<ImagePost url="/images/experiences/badge.jpg" alt="Badge en vivo" />

> _Ejemplo del badge_

Para mejorar la experiencia de usuario, adapte componentes utilizando Tailwind CSS, asegurando una interfaz atractiva y funcional. Implementé la metodología SCRUM para gestionar y desplegar tareas de manera eficiente, optimizando el flujo de trabajo y mejorando la coordinación del equipo.

También configuré y mantuve servidores utilizando Nginx, y migré sitios web a Digital Ocean, lo que mejoró significativamente la escalabilidad y eficiencia del sistema. En cuanto a la publicidad, integré DoubleClick for Publishers (DFP) con Piano y mantuve los módulos publicitarios, asegurando una monetización efectiva del contenido.

Desarrollé APIs utilizando tecnologías como Express, TypeScript, Jest, Husky y Prettier, garantizando un código limpio y bien estructurado. Además, implementé el sistema de gestión de contenido sin cabeza (Headless CMS) Directus, optimizando los procesos empresariales de la compañía y permitiendo una gestión de contenido más flexible y eficiente.
